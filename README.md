## Prepare Environment

You will need [Python], 3 is recommended.
It's also recommended to use [virtualenvwrapper] or [pyenv] to create a
project-level environment.

```shell
pip install -r requirements.txt
```

## Run Monitor

```shell
python -m monitord
```
